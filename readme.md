# STAL (Screen Time Analytics - Self-Hosted Local)

**STAL** is a privacy-first browser extension and local web dashboard that tracks which domains you visit and for how long. Unlike cloud-based trackers, STAL keeps all your data strictly on your machine.

The local web dashboard provides intuitive weekly reports, calculates "focus scores," and visualizes browsing trends using D3.js/Chart.js. You can easily export your data to local JSON files for backup or import it across devices.

## 🚀 Installation

### 1. Clone the Repository

```bash
git clone https://github.com/yourusername/stal.git
cd stal
```
